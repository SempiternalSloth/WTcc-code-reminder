Gaijin Store Partner HUD

Ensuring you never miss a chance to use creator codes.

Step 1: Download

Get the latest extension build below. You will need to unzip this file after downloading.

Download Extension (.zip)

Step 2: Installation Guide

1. Access Extensions: Paste chrome://extensions/ into your browser address bar.
2. Enable Developer Mode: Toggle the switch in the top right corner to ON.
3. Load the HUD: Click the "Load unpacked" button and select the unzipped folder.
